# Golive Capital

Golive Capital é uma plataforma de código aberto focada na otimização da gestão financeira para indivíduos e pequenos negócios. Intuitiva e poderosa, permite acompanhar receitas, despesas e investimentos com clareza, proporcionando insights eficientes para uma tomada de decisão mais inteligente.

## Funcionalidades principais

- **Painel de controle intuitivo**  
  Resumo visual da situação financeira com gráficos e indicadores claros.

- **Gestão de receitas e despesas**  
  Organização com categorização de entradas e saídas de dinheiro.

- **Controle de investimentos**  
  Monitoramento do desempenho da carteira de investimentos.

- **Alertas e lembretes**  
  Notificações sobre contas a pagar e eventos financeiros.

- **Relatórios personalizáveis**  
  Geração de relatórios históricos detalhados para análises financeiras.

## Começando

Siga os passos abaixo para clonar e executar o projeto localmente.

### Pré-requisitos

- [Node.js](https://nodejs.org/)  
- Gerenciador de pacotes: **npm** ou **Yarn**

### Instalação

```bash
git clone https://github.com/wagnerbocchi/golive-capital.git
cd golive-capital
npm install      # ou `yarn install`
npm start        # ou `yarn start`
