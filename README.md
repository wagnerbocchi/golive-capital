Golive Capital
Este repositório é um projeto de código aberto focado em otimizar a gestão e organização de finanças pessoais e de pequenos negócios. A ferramenta foi projetada para ser uma plataforma intuitiva e poderosa, permitindo que os usuários acompanhem receitas, despesas, e investimentos de forma eficiente. O principal objetivo é fornecer insights claros para auxiliar na tomada de decisões financeiras mais inteligentes.

Funcionalidades
Painel de controle intuitivo: Visualize rapidamente a sua situação financeira com gráficos e resumos claros.

Gestão de receitas e despesas: Categorize e monitore todas as suas entradas e saídas de dinheiro.

Controle de investimentos: Acompanhe o desempenho de sua carteira de investimentos.

Alertas e lembretes: Receba notificações sobre contas a pagar e outras datas importantes.

Relatórios personalizáveis: Gere relatórios detalhados para analisar seu histórico financeiro.

Começando
Para clonar e executar o projeto em sua máquina local, siga os passos abaixo.

Pré-requisitos
Certifique-se de que você tem o Node.js e o npm (ou Yarn) instalados em sua máquina.

Instalação
Clone o repositório:
git clone https://github.com/wagnerbocchi/golive-capital.git

Navegue até o diretório do projeto:
cd golive-capital

Instale as dependências:
npm install
ou
yarn install

Inicie o servidor de desenvolvimento:
npm start
ou
yarn start

O aplicativo estará disponível em http://localhost:3000.

Contribuição

Passos para contribuir:

Faça um fork do projeto.

Crie uma nova branch para a sua funcionalidade: git checkout -b minha-nova-funcionalidade.

Faça suas alterações e commit as mudanças: git commit -m 'Adiciona nova funcionalidade'.
